## Suggested reading order for Day 3

Read **Chapter 5** in order **§5.1 → §5.8**. The sections build on each other: univariate calculus → partial derivatives and gradients → Jacobians → matrix calculus → identities → autodiff → higher-order terms → Taylor linearization.

**Prerequisites:** comfortable use of **Chapter 2** (vectors, matrices, linear mappings) and **Chapter 3** (norms and inner products), since gradients are defined with respect to inner products and norms.
