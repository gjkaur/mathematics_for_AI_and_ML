## How to use

To simulate data (e.g. $y = \mathbf{w}^\top\mathbf{x} + \varepsilon$ with Gaussian $\varepsilon$), you need **random variables** (1), **distributions** (2), and **sampling** (3). Align notation with *Mathematics for Machine Learning* Chapter 6.
