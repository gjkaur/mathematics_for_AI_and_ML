# Mathematics for AI and ML

Study materials aligned with **[*Mathematics for Machine Learning*](https://mml-book.com/)** by Marc Peter Deisenroth, A. Aldo Faisal, and Cheng Soon Ong (Cambridge University Press). This repository holds **chapter mappings**, **topic notes**, and **worked examples** organized by study day—not a replacement for the textbook.

Download the official PDF from the [companion site](https://mml-book.com/) or your library; the book itself is **not** included here.

---

## What is in this repository

| Folder | Contents |
|--------|----------|
| [`Day01/`](Day01/) | Linear algebra (Ch. 2), analytic geometry (Ch. 3 §3.1–3.4), and data-oriented pointers (Ch. 6, 8, 9). Topic notes `01-`…`15-`, syllabus map [`Day01.md`](Day01/Day01.md), index [`00-INDEX.md`](Day01/00-INDEX.md). |
| [`Day02/`](Day02/)–[`Day08/`](Day08/) | One guide per day: textbook sections for orthonormal bases & matrix decompositions (Day 2), vector calculus (Day 3), probability (Day 4), optimization (Day 5), models & regression (Day 6), PCA & GMM (Day 7), SVM (Day 8). |

Each `DayNN/DayNN.md` file maps that day’s topics to **book chapters and section numbers** from *Mathematics for Machine Learning*.

---

## How to use it

1. Open the day folder you are studying (e.g. `Day01/`).
2. Read `DayNN.md` for the **reading order** and **MML references**.
3. Use `00-INDEX.md` (Day 1) or the numbered topic files for **notes and solved examples**.

Math in the notes uses **LaTeX** (`$...$`, `$$...$$`). On **GitHub**, open the `.md` file in the repository browser (not Raw)—GitHub renders math with MathJax ([docs](https://docs.github.com/en/get-started/writing-on-github/working-with-advanced-formatting/writing-mathematical-expressions)). Locally, use VS Code with a math-capable Markdown preview, Obsidian, Typora, etc.
